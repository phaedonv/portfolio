Hello,
I am a Le Wagon graduate trained on Full Stack Web Development, seeking new opportunities. Being motivated and eager to learn new skills, I always look for new ways to evolve. I like finding creative ways to approach problems through collaboration and team work.

I am really passionate about modern technologies, so I am always ready to learn and use new frameworks, tools or languages! It 's safe to say, that in programming you are always learning!

During the bootcamp, I learned HTML, CSS, JavaScript, SQL databases, built projects with Ruby on Rails, how to use git and GitHub, and deploy on Heroku. Participated in a team of four, that designed, implemented and shipped to production a clone of AirBnB and also a Rails prototype of We Planit!

Le Wagon is a 9-weeks intensive coding bootcamp, focused on web development and more! During the courses I learned:

  • Ruby
  • Ruby on Rails
  • AJAX
  • JavaScript, (ES6)
  • React & Redux
  • HTML5 - CSS3
  • Bootstrap
  • SQL, PostgreSQL
  • Git & Github
  • Heroku
  • Shell on Linux

After finishing the bootcamp, I started developing a Rails project, called Tunes Of Freedom and deployed it on Heroku. The project is basically a music oriented web-app linked with YouTube playlists and an opportunity to implement the knowledge I earned, so to keep the coding alive! I also created other projects, that are public to see on Github and I never stopped practicing my skills.

Feel free to contact me since I am always open to discussion of new opportunities.
